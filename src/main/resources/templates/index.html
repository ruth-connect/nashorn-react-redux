Nashorn

  • Runs Javascript natively on the JVM
  • Available out of the box as part of Java 8
  • Far better performance than the older Rhino JS engine
  • Can run from within a Java application or directly from the command line
  • Can execute Java code from within Javascript
  • So you can have Java code that calls Javascript that calls more Java code
